# 下單系統

## MasterTradePy

當你下載完元富證券的 Python 模組後，使用以下指令安裝報價系統：

```powershell
pip install .\MasterLink_PythonAPI\MasterTradePy\MasterTradePy\64bit\MasterTradePy-0.0.23-py3-none-win_amd64.whl
```

:::tip
在我們使用時，該套件的版本為 `0.0.23`。
:::
